# 廣清宮快速記帳軟體

專為宮廟設計的財務管理系統，提供完整的收支記錄、財務分析和報表功能。

## 🚀 快速開始

### 方法一：增強版本（推薦）⭐
1. 直接打開 `index-enhanced.html` 文件
2. 支援提醒功能和手機PWA安裝
3. 完整的手機端適配

### 方法二：簡化版本
1. 直接打開 `index-simple.html` 文件
2. 基本記帳功能，輕量級版本

### 方法三：開發環境
由於Node.js版本兼容性問題，建議使用以下步驟：

1. **安裝Node.js 16或18版本**（避免使用22版本）
2. **清理並重新安裝依賴**：
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   npm run dev
   ```

## 📱 使用說明

### 新增記錄
1. 點擊「快速記帳」或對應的收入/支出按鈕
2. 選擇記錄類型（收入/支出）
3. 選擇適當的類別
4. 輸入金額和相關資訊
5. 點擊「儲存」完成記錄

### 查看統計
- 主頁面顯示總收入、總支出和結餘
- 下方顯示最近的記錄列表
- 所有資料自動保存在瀏覽器中

### 匯出資料
1. 點擊「匯出報表」按鈕
2. 系統會自動下載CSV格式的資料文件
3. 可用Excel或其他軟體開啟查看

## 🔧 技術架構

### 簡化版本（index-simple.html）
- **前端框架**：Vue 3（CDN版本）
- **UI組件**：原生CSS + 少量Element Plus
- **資料存儲**：localStorage
- **圖表庫**：ECharts（預留）
- **日期處理**：Day.js

### 完整版本（需要構建）
- **前端框架**：Vue 3 + Vite
- **狀態管理**：Pinia
- **路由管理**：Vue Router
- **UI組件**：Element Plus
- **圖表庫**：ECharts
- **日期處理**：Day.js

## 🛠️ 故障排除

### Node.js版本問題
如果遇到Node.js版本兼容性問題：
1. 使用Node.js 16或18版本
2. 或直接使用 `index-simple.html` 版本

### 資料遺失
- 資料保存在瀏覽器的localStorage中
- 清除瀏覽器資料會導致記錄遺失
- 建議定期使用匯出功能備份資料

### 瀏覽器兼容性
- 支援現代瀏覽器（Chrome、Firefox、Safari、Edge）
- 建議使用最新版本瀏覽器以獲得最佳體驗
